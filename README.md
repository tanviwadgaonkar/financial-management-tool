# Financial Management Tool

A full-stack personal finance tracker that helps users monitor expenses, set budgets, and view analytics to understand spending patterns.

## Features
- Add, edit, and delete income and expenses  
- Category-wise spending breakdown  
- Monthly budget planner  
- Charts for spending trends  
- Secure backend API for transactions and analytics

## Tech Stack
- **Frontend:** React.js  
- **Backend:** Python (Flask)  
- **Database:** PostgreSQL  

## How to Run

```bash
# Clone the repo
git clone https://github.com/tanviwadgaonkar/financial-management-tool.git
cd financial-management-tool

# Install backend dependencies
pip install -r requirements.txt

# Run backend (example)
python app.py

# For frontend (if separate)
# npm install
# npm start
d.
